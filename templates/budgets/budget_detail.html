CSV Format Example

Your CSV file should have the following columns:

date,description,amount,category,member

Example:

2024-11-22,Grocery Shopping,-125.50,Groceries,John
XML Format Example

Your XML file should follow this structure:

<transactions>
  <transaction>
    <date>2024-11-22</date>
    <description>Grocery Shopping</description>
    <amount>-125.50</amount>
    <category>Groceries</category>
    <member>John</member>
  </transaction>
</transactions>
OFX/QFX Bank Statement Files

OFX (Open Financial Exchange) and QFX (Quicken) files are bank statement exports:

  • Download from your bank's online banking portal
  • Usually available in Transaction History or Export section
  • Transactions will be automatically categorized based on merchant names
  • Categories can be adjusted after import
